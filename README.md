# Что находится в репозитории?
- *Более-менее* упорядоченное содержимое актуальных для учащихся групп 4к934\* курсов системы электронного обучения (СЭО) МРК
- Скрипт для автоматической конвертации документов Microsoft Office в PDF (более подробная информация находится в самом скрипте)

Стоит отметить, что здесь *нет* файлов из электронных учебно-методических комплексов (ЭУМК) или электронных образовательных ресурсов (ЭОР): если автору репозитория (или кому-то ещё) *вдруг* окажется не лень (вероятность чего крайне мала), под них будут созданы отдельные «архивы».
